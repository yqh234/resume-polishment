# Aesthetic Effects Library

Use this library to choose the visual language for a professional resume webpage. Pick one primary profile and, if useful, one secondary accent. Keep the result credible for recruiters and hiring managers.

## Profiles

### Executive Minimal

- Mood: calm, decisive, senior, trustworthy.
- Use for: management, strategy, finance, consulting, enterprise sales, operations.
- Visual effects: white or near-white background, charcoal text, fine divider lines, restrained navy or graphite accent, compact section spacing, crisp tables, subtle lettercase labels.
- Avoid: oversized hero type, heavy gradients, decorative cards, flashy icons.

### Analytical Clarity

- Mood: precise, structured, evidence-led.
- Use for: data analysis, product analytics, research, business analysis, risk, operations analysis.
- Visual effects: grid-based layout, small metric tiles, cool gray background bands, blue or teal data accents, monospaced micro-labels for metrics, clear hierarchy.
- Avoid: chart decoration without data, vague skill clouds, overly technical density.

### Product Craft

- Mood: modern, user-centered, polished, collaborative.
- Use for: product manager, UX, growth, product operations, customer experience.
- Visual effects: generous but efficient spacing, soft neutral background, accent color for role fit, case-study blocks, compact process chips, before/after impact framing.
- Avoid: portfolio-like illustration overload, playful gradients, startup cliches.

### Technical Precision

- Mood: reliable, capable, engineering-minded.
- Use for: software engineering, AI engineering, DevOps, data engineering, security, QA.
- Visual effects: dark text on light background, code-like metadata rows, system-status accents, clear stack grouping, project cards with problem-method-result structure.
- Avoid: terminal cosplay, hacker neon, excessive dark mode for resumes.

### Warm Professional

- Mood: approachable, steady, service-oriented.
- Use for: HR, education, customer success, marketing, administration, coordination.
- Visual effects: warm off-white background, muted green or slate accent, readable section blocks, human-centered summary, clear collaboration examples.
- Avoid: beige-dominant monotony, informal scrapbook styling, decorative illustrations.

## Composition Rules

- Use a one-page resume mindset even if the HTML scrolls.
- Put candidate positioning, role target, and strongest match evidence above the fold.
- Use section labels that sound like resume sections, not marketing pages.
- Treat every decorative effect as subordinate to recruiter scanning speed.
- Prefer simple CSS effects: border, shadow under 12px blur, subtle background bands, accent rules, restrained badges, and print-safe colors.

## Copy Tone Modifiers

- "evidence-first": lead with proof and outcomes.
- "role-aligned": mirror the target job's vocabulary without keyword stuffing.
- "measured confidence": sound capable without exaggeration.
- "business-aware": connect work to cost, revenue, efficiency, risk, user experience, quality, or growth.
- "human but formal": readable Chinese or English, no stiff bureaucratic phrasing.
